Neo4j Desktop 的使用

1. 下载.csv文件

2. DBMS创建

  • 打开主界面，在左上角找到+New，点击，然后选择Create project
  • 创建project之后，在右侧找到+Add，点击，然后选择Local DBMS
  • 创建Local DBMS时，我们可以自由更改Name和Password，然后点击Create
  • 点击创建好的DBMS，右侧出现Start字样，点击Star
  • 在页面顶部找到open，然后点击进入neo4j网站

3. .csv文件的导入

  • 在Neo4j Desktop主界面找到刚刚创建的DBMS点击，末尾出现···，点击···，选择Open folder，再选择Import，进入存放.csv 文件的文件夹下

  • 将关于数据集的.csv文件复制粘贴在Import文件夹下

  • 打开neo4j网页，左侧第一项为数据库信息，可在此找到导入的数据（标签）（关系）等，右侧为数据导入及知识图谱的展示

  • 点击右侧顶部neo4j$,在此处进行编写

  • 节点导入

    • 输入代码

      LOAD CSV WITH HEADERS FROM “file:///文件名.csv” AS line

      MERGE(p:标签名{表中的某一项表头名:line.表中的某一项表头名······})

    • 注意：MERGE也可以改成CREATE，但是CREATE不会去判断节点中是否有重复数据，只是添加；MERGE等于是CREATE+MATCH，如果节点中有重复数据，就不会添加进去

    • 表中的某一项表头名:line.表中的某一项表头名 可以改为表中的某一项表头名:line.[x] x为那一项表头名所在列数

  • 关系导入

    • 输入代码

      LOAD CSV WITH HEADERS FROM “file:///文件名.csv” AS line
      MATCH(from:某个标签(某一项表头名：line.某一项表头名}),(to:另一个标签(另一项表头名:line.另一项表头名)})
      MERGE(from)-[r：关系名（某种关系：line.某种关系}]->（to)

    • 同样的MERGE也可以用CREATE代替

    • 代码的第二行表示：将两个标签的节点的联系建立起来
      代码的第三行表示：这两个标签的节点的联系是通过某种关系建立起来的
